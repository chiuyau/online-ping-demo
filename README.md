# online-ping-demo

简易在线ping工具，将ping命令输出实时显示到web页面

- python 版本基于(flask | flask-socketio | eventlet)
- nodejs 版本基于(express | socket.io)


### Usage

下载示例

```shell
git clone git@github.com:sincerefly/online-ping-demo.git
cd online-ping-demo
```

启动应用

*Python3*

```bash
cd python3

# 安装依赖
pip3 install -r requirements.txt

# 启动
python3 app.py
```

*Node.js*

```bash
cd nodejs

# 安装依赖
npm install

# 启动
node app.js
```

打开浏览器访问 5000 端口查看
